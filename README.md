# Simple Traditional Chatbot

## Overview
This project implements a rule-based (non-LLM) chatbot using keyword-based intent detection.

## Development Environment
Developed using Databricks Community Edition.

## Features
- Greeting recognition
- Capability listing
- Working hours response
- Help instructions
- Fallback handling for unknown inputs

## Architecture
User Input → Intent Detection → Response Mapping → Output

## How to Run
Run chatbot.py using Python 3.8 or higher.
